# 일본어 복습 플랫폼

모바일 브라우저에서 교재 사진을 올려 단어, 한자, 형변환, 문법, 예문을 복습 카드로 저장하는 Next.js 앱입니다. 데이터는 로그인 없이 현재 브라우저에 저장됩니다.

## 실행

```bash
npm install
npm run dev
```

사진 분석을 쓰려면 `.env.local`에 `OPENAI_API_KEY`를 넣습니다. 공개 배포에서는 다른 사람이 내 API 비용을 쓰지 못하도록 `UPLOAD_PIN`도 설정하는 편이 좋습니다.

## Codex 사진 입력

앱 안에서 사진을 올릴 수도 있고, Codex 채팅에 교재 사진을 올린 뒤 “일본어 복습 앱 import JSON으로 바꿔줘”라고 요청해도 됩니다. Codex가 만들어준 JSON은 앱의 설정 화면에서 가져올 수 있습니다.

## 공개 학습장

다른 사람이 같은 자료를 볼 수 있게 하려면 앱에서 백업 JSON을 내보낸 뒤 `public/study-feed.json`을 그 내용으로 바꾸고 다시 배포합니다. 방문자는 처음 들어올 때 공개 학습장을 자동으로 받고, 복습 진도는 각자 브라우저에만 저장됩니다.

## 배포

Vercel에 배포할 때 사진 분석을 켜려면 프로젝트 환경변수에 `OPENAI_API_KEY`, `OPENAI_MODEL`, `UPLOAD_PIN`을 설정합니다. 공개 학습장과 브라우저 복습만 쓸 경우에는 환경변수 없이도 앱이 동작합니다.
